# ISOMan

ISOMan is a self-hosted Linux ISO manager for Cloudron.

It lets you queue ISO downloads, track progress in real time, verify checksums, and serve files through a built-in HTTP directory view.

## Highlights

- Concurrent download queue with progress updates
- SHA256 / SHA512 / MD5 checksum verification
- Organized storage under `/app/data/isos`
- Embedded web UI and API in one Cloudron app
